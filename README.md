# 💳 Billetera Digital

Una aplicación web moderna de gestión financiera construida con HTML5, CSS3, Bootstrap y JavaScript.

## 🌟 Características

✅ **Autenticación** — Login seguro con validación
✅ **Depósitos** — Realizar depósitos con saldo actualizado
✅ **Transferencias** — Enviar dinero a contactos
✅ **Historial** — Ver últimas transacciones
✅ **Responsive** — Diseño adaptable a móviles
✅ **Almacenamiento Local** — Los datos se guardan en el navegador

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, Bootstrap 5
- **JavaScript**: Vanilla JS + jQuery
- **Almacenamiento**: localStorage
- **Control de versiones**: Git
